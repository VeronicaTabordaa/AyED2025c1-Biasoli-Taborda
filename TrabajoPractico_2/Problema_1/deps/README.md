# 📑Dependencias del proyecto

Registro de los módulos necesarios para que la aplicación pueda ser ejecutada. Las dependencias necesarias se encuentran en el archivo `requirements.txt`

En el código del proyecto se usan únicamente módulos de la biblioteca estándar de Python,
por lo que no se requieren librerías externas para este proyecto, de manera que el archivo
`requirements.txt` se encuentra vacío
